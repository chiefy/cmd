edit

Edits a command

$ cat ./script | ssh alpha.cmd.io :edit <name> -

:edit will rebuild the command <name> with the script passed via STDIN. The builtin requires the - second argument to inform it to read from STDIN, as future versions may introduce an interactive mode.

See :create for the expected script format.
